# Bienvenido al laboratorio TechLogistics

En este laboratorio aprenderás a:

- Implementar almacenamiento tipo S3 usando MinIO.
- Configurar AWS CLI.
- Crear buckets.
- Automatizar backups.
- Simular migraciones cloud.

## Objetivo

La empresa TechLogistics necesita automatizar el respaldo de archivos críticos hacia almacenamiento cloud.
